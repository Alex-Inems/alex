## ⚡ Quick Start

### 1. Prerequisites
Ensure you have **Python 3.10+** installed on your system.

### 2. Environment Configuration
To leverage the LLM scoping engine, make sure your Gemini API key is configured. The CLI automatically searches for environment variables in the following sequence:
1. Environment variables in the active terminal session.
2. `.env.local` in the project root.
3. `.env` in the project root.

To add your key, simply add the following line to your `.env.local`:
```env
GEMINI_API_KEY=your_actual_gemini_api_key_here
```

> [!WARNING]
> Never commit `.env` or `.env.local` files to Git. They are explicitly added to your project's `.gitignore` to prevent sensitive credential leaks.

### 3. Running the AI Workflow Builder

You can trigger the AI Builder immediately from your terminal:

```bash
# Generate a scraper automation script
python automation/ai_workflow_builder.py "Scrape competitor price listings and save to excel sheet" -o scrapers/competitor_pricing.py

# Generate a lead enrichment pipeline
python automation/ai_workflow_builder.py "LinkedIn lead generation email finder and HubSpot CRM synchronization" -o sales/lead_pipeline.py
```

### 4. Running Offline / Local Fallback
If you do not have an API key or internet access, you can run the builder in local heuristic mode:
```bash
python automation/ai_workflow_builder.py "Customer support ticket router" -o tickets/support_router.py --local
```

## 🛡️ Production-Grade Coding Standards

Every script output by our generator follows strict systems-engineering principles:
* **Strict Type Hints**: Utilizes standard Python typing (`List`, `Dict`, `Any`, `Optional`) to ensure IDE auto-completion and static analysis checks.
* **Structured Logging**: Implements standard `logging` instead of fragile `print` statements, keeping logs clean and compatible with cloud monitors (AWS CloudWatch, Datadog).
* **Exception Handling**: Features nested `try-except` blocks with precise HTTP and connection handler wrappers.
* **Security First**: Pulls API configurations securely using `os.getenv` rather than hardcoding credentials in the source code.

## 🚀 Scaling to Agentic Architectures

When transitioning your scoped pipelines into complex, multi-agent orchestrations, we recommend incorporating:

### CrewAI (Multi-Agent Workforces)
```python
from crewai import Agent, Task, Crew, Process

# Define a research agent
researcher = Agent(
    role='Market Analyst',
    goal='Scrape and analyze pricing structures',
    backstory='Elite data collector specializing in SaaS competitor metrics.',
    verbose=True
)
```

### LangChain (Semantic Retrieval)
```python
from langchain_google_genai import ChatGoogleGenerativeAI
from langchain_core.prompts import ChatPromptTemplate

# Create conversational agent pipeline
llm = ChatGoogleGenerativeAI(model="gemini-2.5-flash")
prompt = ChatPromptTemplate.from_template("Optimize this business email: {email}")
chain = prompt | llm
```
